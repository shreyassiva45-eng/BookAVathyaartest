## Overview
Book A Vaathyaar (formerly Purohit Connect) is a modern platform designed to connect users with verified Hindu Priests (Purohits/Vaathyaars) for sacred rituals and ceremonies. 

### Features
- **Priest Discovery**: Find nearest priests based on your location.
- **Dedicated Dashboards**: Separate seamless experiences for Normal Users and Priests.
- **Modern Booking Flow**: Step-by-step intuitive booking process.
- **Cross-Platform**: Modern web application with native Android app integration.
